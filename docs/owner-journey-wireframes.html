6. Design Specifications

Color Palette
Primary Colors:
  • Primary Blue: #2563eb
  • Primary Dark: #1e40af
  • Primary Light: #dbeafe
Neutral Colors:
  • Text Primary: #1e293b
  • Text Secondary: #64748b
  • Text Muted: #94a3b8
  • Border: #e2e8f0
  • Background: #f8fafc
Status Colors:
  • Success: #10b981 (Active, Confirmed)
  • Warning: #f59e0b (Pending, Review)
  • Error: #ef4444 (Inactive, Cancelled)
  • Info: #3b82f6 (Verified, Featured)
Typography
Font Family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Arial, sans-serif

Font Sizes:
  • H1: 32px (Page Titles)
  • H2: 24px (Section Headers)
  • H3: 20px (Card Titles)
  • Body: 14px (Regular Text)
  • Small: 13px (Labels, Captions)
  • Tiny: 12px (Badges, Tags)
Component Specifications
Buttons:
  • Height: 48px (Primary), 40px (Secondary)
  • Border Radius: 6px
  • Padding: 0 24px
  • Font Weight: 600
Input Fields:
  • Height: 45px
  • Border: 2px solid #e2e8f0
  • Border Radius: 6px
  • Focus: border-color #2563eb
Cards:
  • Border: 2px solid #e2e8f0
  • Border Radius: 8px
  • Padding: 20px
  • Box Shadow: 0 1px 3px rgba(0,0,0,0.1)
