6. Integration Points

Third-Party Services
Maps: Google Maps API (Street View, Directions)
Messaging: WhatsApp Business API
Payment: Razorpay / Stripe for commission payments
SMS/OTP: Twilio / AWS SNS
Email: SendGrid / AWS SES
Storage: AWS S3 / Cloudinary for image uploads
Search: Algolia / Elasticsearch for advanced search

7. Key Metrics to Track

Metric Description Target
User Registration Rate % of visitors who create accounts 15-20%
Search to View Rate % of searches that lead to PG detail views 40-50%
View to Contact Rate % of views that lead to contact/visit request 20-30%
Property Approval Time Average time to approve new properties < 24 hours
Owner Retention % of owners active after 6 months 70%+
Payment Success Rate % of commission payments received on time 95%+
