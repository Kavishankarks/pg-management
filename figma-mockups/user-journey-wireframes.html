1. Landing Page & Browse

User Flow Overview
Entry Points:
  • Direct URL: bookpg.com
  • Search engine results
  • Social media/ads
Flow Steps:
  1. Land on homepage with hero section
  2. Browse featured PGs or search
  3. Apply filters (location, price, gender, amenities)
  4. View PG listing cards
  5. Click to view details
  6. Schedule visit or contact owner
  7. Login/Signup to book

5. Design Specifications

Color Palette
Primary Colors:
  • Primary Blue: #2563eb
  • Primary Dark: #1e40af
  • Primary Light: #dbeafe
Neutral Colors:
  • Text Primary: #1e293b
  • Text Secondary: #64748b
  • Text Muted: #94a3b8
  • Border: #e2e8f0
  • Background: #f8fafc
Status Colors:
  • Success: #10b981 (Available, Confirmed)
  • Warning: #f59e0b (Limited, Pending)
  • Error: #ef4444 (Unavailable, Cancelled)
  • Info: #3b82f6 (Verified, Featured)
Rating:
  • Star Gold: #f59e0b
Key User Interactions
Search Behavior:
  • Autocomplete on 3+ characters
  • Show recent searches
  • Voice search support (mobile)
  • Natural language processing
Filtering:
  • Applied filters shown as chips
  • One-click to remove filter
  • Filter count badge
  • Save filter preferences
Map Integration:
  • Blurred/hidden until login
  • Show only after authentication
  • Google Maps embed with markers
  • Street view integration
  • Directions link
WhatsApp Integration:
  • Pre-filled message template
  • Include PG name and link
  • Owner's WhatsApp number
  • Track clicks for analytics
Responsive Breakpoints
  • Mobile: 320px - 767px (Single column)
  • Tablet: 768px - 1023px (2 columns for cards)
  • Desktop: 1024px+ (3-4 columns for cards)
  • Navigation: Hamburger menu below 768px
